# 1、Visual Studio 2019快捷键

```
Ctrl + K + D 格式化代码
Ctrl + K + C 单行注释
Ctrl + K + U 取消注释
Ctrl + R + E 生成 属性get set方法
Shift + Alt + F10 / Alt + Enter 快速导入引用
Ctrl + R + G 删除无效Using
Ctrl + J 强制代码提示
Ctrl + G 跳转到指定行
cw + Tab + Tab 生成代码Console.WriteLine()
ctor + Tab + Tab，生成无参构造器
Ctrl + E + E 在交互窗口中运行选中代码
Ctrl + H 批量替换
Ctrl + Delete 删除光标右侧的所有字
Ctrl + Shif + L 删除光标所在行
Ctrl+Shift+F=在文件中查找
Ctrl + Shift + F=查找所选条目在整个解决方案中的引用
Ctrl + Shift + Space（空格键） 查看方法文档
F1  查看MSDN文档

F12 = 转到定义
Shift+F12=查找所有引用
```
